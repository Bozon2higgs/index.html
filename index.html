Guide utilisateur — Comment remplir ce formulaire

Vous n’avez pas besoin d’être juriste. Écrivez avec vos mots. Ce guide est là pour vous aider.

1. Identité

Choisissez Anonyme si votre sécurité l’exige. Choisissez Identifié si vous souhaitez apparaître.

2. Pays et dates

Indiquez le lieu des faits et les dates connues. Les estimations sont acceptables.

3. Victimes

Décrivez qui est affecté : civils, groupes, populations vulnérables.

4. Auteurs présumés

Indiquez-les si vous les connaissez. Sinon, laissez vide.

5. Violations

Cochez ce qui correspond le mieux, même approximativement.

6. Exposé des faits (résumé)

Expliquez brièvement ce qui s’est passé : quoi, où, quand.

7. Faits détaillés

Décrivez librement : chronologie, témoignages, conséquences humaines.

8. Preuves

Listez ce qui existe (photos, vidéos, articles, rapports). HFH ne téléverse rien.

9. Demandes

Expliquez ce que vous demandez : enquête, protection, suivi.
