# HFH — Human For Human

HFH (Human For Human) — это простой, безопасный и локальный инструмент для структурирования сообщений о нарушениях прав человека без сбора данных, серверов и автоматической передачи.

HFH — это инструмент.  
Это не платформа, не организация и не орган власти.

## Цель

Помочь любому человеку структурировать чувствительную информацию в условиях стресса, срочности, наблюдения или ограниченного доступа к интернету.

HFH не интерпретирует, не оценивает и не передаёт данные.

## Основные принципы

- отсутствие сбора данных
- отсутствие передачи данных
- отсутствие backend
- отсутствие учетных записей
- отсутствие cookies и аналитики

Вся работа происходит локально в браузере.

## Функции

- структурированная форма
- локальный экспорт в PDF
- локальный экспорт в редактируемый DOCX
- многоязычный интерфейс
- поддержка мобильных устройств

## Безопасность и конфиденциальность

Все данные остаются на устройстве пользователя.

HFH можно использовать офлайн после загрузки и через Tor.

## Чем HFH не является

HFH не является платформой для подачи жалоб или системой передачи данных.

Он лишь помогает структурировать информацию.

## Статус проекта

HFH стабилен, функционален и готов к версии 1.0.
